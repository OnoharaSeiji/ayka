# Scripts

## gerar_apostila_dia.py
Gera ou regenera um arquivo diário a partir da base JSONL.

Exemplo:
```bash
python scripts/gerar_apostila_dia.py --date 2026-04-01
```

## organizar_base.py
Recria os índices mensais e regenera todos os arquivos-base.

Exemplo:
```bash
python scripts/organizar_base.py
```
